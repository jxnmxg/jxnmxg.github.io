1. Who shared the malware samples?

After logging into the platform and reviewing the email, the individual who shared the malware samples was: Oliver Bennett.

2. What is the SHA1 hash of the file “pRsm.dll” inside samples.zip?

Use the command sha1sum in terminal:

$ sha1sum pRsm.dll

3. Which malware framework utilizes these DLLs as add-on modules?

Research points to MgBot as the malware framework that uses these DLLs as add-on modules.

4. Which MITRE ATT&CK Technique is linked to using pRsm.dll in this malware framework?

Reading the same article as the one I provided about i found out the associated MITRE ATT&CK Technique is T1123.
